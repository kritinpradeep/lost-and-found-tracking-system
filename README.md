# Lost & Found Item Tracking System

## 📌 Overview
This project is a full-stack web application designed to help users report, search, and recover lost items efficiently within a campus or organization.

## 🚀 Features
- User authentication (Login/Signup)  
- Report lost and found items with images  
- Search and filter items  
- Smart matching system  
- Contact/notification system  

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Python (Flask)  
- Database: SQLite  
- DevOps: Docker, GitHub Actions, Kubernetes  

## ⚙️ Architecture
- REST-based Flask backend  
- SQLite database for item and user data  
- Containerized deployment using Docker  
- CI/CD pipeline using GitHub Actions  

## 🧪 Testing
- Unit Testing  
- Integration Testing  
- System Testing  
- Acceptance Testing  

## ▶️ How to Run
1. Clone the repository  
2. Install dependencies  
3. Run Flask server  
OR  
4. Run using Docker
